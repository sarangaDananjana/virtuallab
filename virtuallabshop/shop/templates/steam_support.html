Game Installation Guide

Follow these steps carefully to set up your game correctly.

1

Use a Dedicated Steam Account

This is the most important step for your account's safety.

CRITICAL WARNING: Protect Your Main Steam Account

The games you receive from us use a patch. While we ensure our files are safe, any modification to game files carries a small risk. If something goes wrong, you could lose access to games you have purchased on Steam.

  • We will always replace games you bought from us for free.
  • We CANNOT replace games you bought from the official Steam store.

To protect your library, ALWAYS use a separate Steam account for games from Virtual Lab. This can be a new account or an old one you don't use and haven't spent money on.

How to Create a New Steam Account

  1. Go to the official Steam website: store.steampowered.com/join
  2. Enter your email address and complete the registration form.
  3. Verify your email address by clicking the link Steam sends you.
  4. Choose a new account name and password.
  5. Log out of your main Steam account and log in with your new one before proceeding.
2

Install Steam Tools

This tool is required to apply the game patches.

  1. Download: Find the `SteamTools.zip` file on DVD 1 provided with your order.
  2. Extract: Unzip the file to a memorable location, for example, `C:\SteamTools`.
  3. Run Setup: Open the extracted folder and run `Setup.exe`. Follow the on-screen instructions to complete the installation.
3

Add Manifest Files

These files tell Steam Tools which games to patch.

  1. Locate Manifests: On DVD 1, find the `Manifests` folder. Inside, you will see files ending in `.acf`.
  2. Find SteamApps Folder: Navigate to your Steam installation directory. Inside, find the `steamapps` folder. (e.g., `C:\Program Files (x86)\Steam\steamapps`).
  3. Copy Files: Copy all the `.acf` files from the DVD's `Manifests` folder directly into your `steamapps` folder.
4

Copy Game Files

The final step is to place the game data where Steam can find it.

  1. Open DVDs: Insert the game DVD(s) from your order. You will see a folder with the game's name.
  2. Navigate to `common`: Go back to your `steamapps` folder. Inside, open the `common` folder.
  3. Copy Game Folder: Copy the entire game folder from the DVD and paste it inside the `common` folder. This may take some time depending on the game size.
5

Enjoy Your Game!

You're all set and ready to play.

Restart Steam completely. Once it relaunches, you will see the game in your library, ready to be launched. If you run into any issues, please open a support ticket.
